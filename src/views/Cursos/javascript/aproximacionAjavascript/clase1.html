Qué es JavaScript?

JavaScript es el lenguaje de programación que te permitirá crear aplicaciones web y móviles a lo largo de múltiples tipos de dispositivos y navegadores web. Esta es una definición muy general de este lenguaje. Y si hablamos en un nivel más técnico, encontrarás que es un lenguaje que sigue la especificación ECMAscript y que es, junto a HTML y CSS, una de las tres tecnologías elementales en el mundo de la web. Pero la mejor forma de entender JavaScript es viéndolo en acción. En primer lugar, donde encontrarás JavaScript es en cualquier web. Y, aunque el alcance de este lenguaje llega también a dispositivos móviles y otro tipo de "hardware", la web siempre es el punto clave de partida. Lo que ves en una página web es la combinación de este tridente de tecnologías: HTML, CSS y JavaScript, interpretados por el navegador web que estés usando como en este momento. La primera capa es HTML, la cual solo da estructura y significado al contenido que se muestre. Este, al ser un lenguaje estrictamente de marcado, carece de ciertas características de vista e interactividad. Para cuestiones de vista, existe la segunda capa en una página web, la cual es cubierta por CSS y, aunque también puede disponer de algunas opciones de interactividad, se enfoca principalmente en cómo se verá el contenido, es decir, en su presentación. Ya estructurado y bien presentado el contenido, entonces entra en acción la tercera capa, que es cubierta por JavaScript, un lenguaje interpretado que se ejecuta en el navegador e interactua perfectamente con las capas de HTML y CSS manipulando e interactuando con el contenido. Desde el más básico clic de un botón hasta una sofisticada animación o la invocación de datos y servicios remotos. JavaScript se ha vuelto la piedra angular en el desarrollo web contemporáneo. La evolución de la tecnología ha llevado a extender su uso más allá de un navegador, en el Escritorio y hacia otros dispositivos, garantizando así su forma de trabajo y ejecución. ¿Dónde? Siguiendo el orden de las capas que hemos mencionado, se estructura primero el contenido con HTML, después se dispone de presentación con las reglas expuestas con CSS y, finalmente, se le da vida con la interactividad de la que dispone JavaScript. Este lenguaje ha evolucionado maravillosamente desde que fue creado en diciembre de 1995 por Brendan Eich, agregando y mejorando características y elementos que hacen que sea cada vez más fácil y sencillo programar con JavaScript. Y, por cierto, cuidado en no confundir JavaScript con Java, que, si bien ambos son lenguajes de programación, tienen entornos y contextos muy distintos; aunque ahora la sintaxis en ambos tenga muchas similitudes. Lo que sí es muy cierto: el alcance y poder de JavaScript es muy amplio, por lo cual existen ya muchas plataformas y tecnologías donde este lenguaje es su base fundamental, como el caso de Node o algunos "frameworks" muy poderosos e importantes como Angular, React, o Vue, y algunas tecnologías móviles como Native JavaScript o Ionic, entre otras. Pero, vamos, esto solo es un preámbulo de lo que es este lenguaje. Comencemos desde los fundamentos más elementales para que puedas sacar el mayor provecho de este lenguaje y conozcas muy bien todas sus capacidades.
